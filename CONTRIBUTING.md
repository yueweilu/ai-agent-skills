# Contributing to Industry Skills Collection

Thank you for your interest in contributing! We rely on community knowledge to make this collection accurate and comprehensive.

## How to Contribute

### 1. Add a New Industry

If your industry is missing, please add it!

1.  Create a new Markdown file in the `skills/` directory. Use the naming convention `industry-name.md` (e.g., `digital-marketing.md`).
2.  Use the following template:

    ```markdown
    # [Industry Name]

    ## Core Concepts
    - Concept 1: Description...
    - Concept 2: Description...

    ## Tools & Technologies
    - Tool A
    - Tool B

    ## Soft Skills
    - Skill X
    - Skill Y
    ```

3.  Link your new file in the main `README.md`.
4.  Submit a Pull Request.

### 2. Update Existing Skills

Found a typo? Missing a critical tool?

1.  Open the relevant file in `skills/`.
2.  Edit the content.
3.  Submit a Pull Request.

## Guidelines

- **Keep it clear**: Use simple language.
- **Be objective**: Focus on widely recognized skills and tools.
- **Categorize**: Group skills logically (e.g., "Beginner", "Advanced" or "Tools", "Concepts").

We look forward to your contributions!
